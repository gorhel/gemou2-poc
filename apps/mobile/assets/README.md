# Assets Machi

## Logo et icônes

### Fichiers SVG (vectoriels)
- `logo.svg` - Logo complet avec texte "Machi" et slogan "Trouve ton game" (400x200px)
- `logo-full.svg` - Version complète du logo (400x200px)
- `logo-icon.svg` - Version icône seule sans texte (200x200px)

### Fichiers PNG requis
Les fichiers PNG doivent être générés depuis les SVG. Voir `generate-png-assets.md` pour les instructions.

- `icon.png` - 1024x1024px (icône principale iOS/Android)
- `adaptive-icon.png` - 1024x1024px (icône adaptative Android)
- `splash.png` - 1284x2778px (splash screen avec fond #6366F1)
- `favicon.png` - 48x48px (favicon web)

## Génération des PNG

Pour générer les fichiers PNG depuis les SVG, consultez `generate-png-assets.md` qui contient :
- Instructions pour convertisseurs en ligne
- Commandes ImageMagick
- Instructions pour éditeurs graphiques

## Palette de couleurs

- Primaire : #6366F1 (Indigo)
- Secondaire : #8B5CF6 (Violet)
- Accent : #F59E0B (Ambre)
- Neutre : #F0F2F5 (Gris clair)
- Texte : #1F2937 (Gris foncé)
